# Gender controlled vocabulary

Returns controlled vocabulary options for organism sex or gender.

## Usage

``` r
gender_vocabulary()
```

## Value

A character vector of gender options

## Details

Provides standardised categories for biological sex of sampled
organisms. Options include:

Not reported

Not relevant

Male

Female

Mixed

Hermaphrodite

Other
