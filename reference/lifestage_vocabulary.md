# Life stages controlled vocabulary

Returns controlled vocabulary options for organism life stages.

## Usage

``` r
lifestage_vocabulary()
```

## Value

A character vector of life stage options

## Details

Provides standardised categories for developmental stages of sampled
organisms. Options include:

Not reported

Not relevant

Adult

Juvenile

Larva

Embryo

Egg

Seedling

Mature

Young

Mixed

Other
