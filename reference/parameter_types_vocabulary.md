# Parameter types controlled vocabulary

Returns controlled vocabulary options for broad parameter
classifications.

## Usage

``` r
parameter_types_vocabulary()
```

## Value

A character vector of parameter type options

## Details

Provides high-level categories for classifying measured parameters.
Options include:

Not relevant

Stressor

Quality parameter

Normalisation

Background

Ecological Indicator

Other
