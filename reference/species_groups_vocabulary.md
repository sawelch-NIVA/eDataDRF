# Species groups controlled vocabulary

Returns controlled vocabulary options for broad taxonomic groups from
EPA ECOTOX database.

## Usage

``` r
species_groups_vocabulary()
```

## Value

A character vector of species group options

## Details

Provides high-level taxonomic groupings used in the ECOTOX database.
Options include:

All

Algae

Amphibians

Bacteria

Birds

Crustaceans

Ecosystem

Fish

Fungi

Insects/Spiders

Invertebrates

Mammals

Molluscs

Moss/Hornworts

Plants

Reptiles

Worms

Other
