# Tissue types controlled vocabulary

Returns controlled vocabulary options for biological tissue types.

## Usage

``` r
tissue_types_vocabulary()
```

## Value

A character vector of tissue type options

## Details

Provides standardised categories for biological tissues sampled from
organisms. Options include:

Not reported

Not relevant

Whole body

Total soft tissues

Muscle

Liver

Kidney

Fat/Adipose

Skin

Bone

Pyloric caeca

Body wall

Brain

Heart

Lung

Gill

Gonad

Shell

Carapace

Digestive Gland

Mantle

Blood

Egg

Larva

Leaf

Root

Stem

Fruit

Seed

Other

In future, this will be standardised with the BRENDA and/or Uberon
tissue ontologies.
